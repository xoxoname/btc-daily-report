# BTC Daily Report Bot

비트코인 실시간 수익 리포트 및 급변동 감지 알림 시스템

## 기능

- Coinbase 실시간 BTC 가격 모니터링
- Bitget API를 통한 포지션 수익률 계산
- 매일 오전 9시 / 오후 1시 / 오후 11시 자동 리포트 전송
- 5분 간격 급변동 감지 시스템
- 트럼프 발언, 미국 지표, 증시 영향 등 GPT 기반 이슈 요약
